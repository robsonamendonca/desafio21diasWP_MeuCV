== Installation ==

Install Responsive Meu CV either via the WordPress.org plugin directory or by uploading the files to your server.

After activating Meu CV, go to plugin menu.

Click Add Meu CV and upload image / poster slides.

Publish the gallery and copy Meu CV shortcode from the bottom of the add images box and embed shortcode on any Page/Post/Text Widget.

That's it. You're ready to go!
